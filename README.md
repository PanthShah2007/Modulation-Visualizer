# 📡 Modulation Visualizer

> **An Interactive Analog Communication Learning Tool for Electronics & Communication Engineering Students**

A modern, interactive web application designed to help students understand the concepts of **Amplitude Modulation (AM)**, **Frequency Modulation (FM)**, and **Phase Modulation (PM)** through real-time waveform visualization.

Instead of relying only on textbook diagrams, this application allows users to experiment with different modulation parameters and instantly observe how the carrier signal changes. It is built entirely using **HTML, CSS, and JavaScript**, making it lightweight, responsive, and easy to use.

# 📖 About the Project

Modulation is one of the most fundamental concepts in Analog Communication. Although the mathematical theory is straightforward, many students find it difficult to visualize how the message signal modifies the carrier wave.

This project bridges that gap by providing an interactive environment where users can change various parameters and immediately see the resulting waveforms.

The application is intended for:

* Electronics & Communication Engineering students
* Beginners learning Analog Communication
* Teachers and educators
* Engineering laboratories
* Self-learning enthusiasts

# ✨ Features

## 📈 Interactive Waveform Visualization

Visualize all three signals simultaneously:

* Message Signal
* Carrier Signal
* Modulated Signal

All waveforms update instantly whenever a parameter changes.

---

## 📡 Three Modulation Modes

Switch between:

* Amplitude Modulation (AM)
* Frequency Modulation (FM)
* Phase Modulation (PM)

Each mode uses its corresponding mathematical equation and visualization.

Modify:

* Message Frequency
